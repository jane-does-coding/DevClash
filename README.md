## Run Locally

Clone the project

```bash
  git clone https://github.com/jane-does-coding/cooking-website
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm i
```

Add .env

```bash
  DATABASE_URL=
  NEXTAUTH_SECRET=
```

Start the server

```bash
  npm run dev
```
